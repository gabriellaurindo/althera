# Althera Mod

A summoning-based mod featuring custom entities powered by GeckoLib.

## Features
- Mana system
- Custom summon mechanics
- Animated 3D entities

## Dependencies
- NeoForge 1.21.1
- GeckoLib (4.7+)

## How to run (dev)

    ./gradlew runClient

**NeoForge 1.21.1 - 21.1.219**
